# CK3-History-GUI
Making a Unity Based GUI for CK3 History Extractor by TCA166

## Basics
I created this program while getting back in to Unity development after a few years to freshen up my memory of the engine. I picked this project because it was a simple to create a basic GUI and since I use CK3 history extractor software quite frequently, I figured why not make a gui for my ease of use.

It does not have many functions except just picking a folders and printing the output. And thats it.

I may add features in the future such as opening the log file.

## Updating
To update the executable, replace the executable at [CK3 History Extractor_Data\StreamingAssets\CK3 Extractor\binary]

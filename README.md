# CK3-History-GUI
Making a Unity Based GUI for CK3 History Extractor by [TCA166](https://github.com/TCA166/CK3-history-extractor).

# Introduction
I created this program while getting back in to Unity development after a few years to freshen up my memory of the engine. I picked this project because it was a simple to create a basic GUI and since I use CK3 history extractor software quite frequently, I figured why not make a gui for my ease of use.

It does not have many functions except just picking a folders and printing the output. And thats it.

I may add features in the future such as extracting the repo and compile it from the GUI.

I will be adding the ability to link mod folders and dump the data when I get the time.

Localization does not seem to be working so I have disabled it for now.

# Help

Save File - Choose your save file (in ck3. format)
Game Folder - Select your game folder. Basically the Crusder Kings III folder
Output Folder - Choose where you want the data be stored. It will create a new folder in the format : {Steam Username}'s History. If that folder exists then it will overwrite it

Language - Not Implemented. 
Depth - Descide how much of historied is to be rendered. (Current Limit is 4, set by me might add more after testing, the higher it is the longer it will take to load)

Run - Runs the executable

Log - If this pops up, it means something went wrong. I Would advise you to click on it and then send the log TCA166 git. The Thrid button (The one that looks like a White 'T' on black background) will send you to the repo.

# How to update the executable

The Update button opens the folder with the executable, just copy the new ck3_History_Extractor.exe there. Normally I would have the updated the binary, but in case I havent or you want to use an older version.
